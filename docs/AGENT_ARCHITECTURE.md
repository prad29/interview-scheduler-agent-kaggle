# Agent Architecture

## Overview

The system uses a multi-agent architecture with specialized agents.

## Agents

1. **Resume Parser Agent** - Extracts structured data from resumes
2. **Skills Matcher Agent** - Matches candidate skills with job requirements
3. **Cultural Fit Agent** - Analyzes cultural alignment
4. **Interview Scheduler Agent** - Automates interview scheduling
5. **Orchestrator Agent** - Coordinates all agents
