# Intelligent Recruitment & Talent Matching System

An enterprise-grade agentic AI solution built with Google ADK for automated candidate evaluation and interview scheduling.

## Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Copy `.env.example` to `.env` and configure
3. Run setup: `python scripts/setup_calendar_auth.py`
4. Start the application: `python api/main.py`
